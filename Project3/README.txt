Con el script descargador.sh descargamos cada archivo.csv desde el servidor entregado (https:
//srvbioinf1.utalca.cl/~fduran) para su uso.
Luego con el script separador.sh extraemos los datos a utilizar desde los archivos.csv antes descargados, y los organizamos en otros .csv para facilitar su uso.
Con el promediador.sh se debería poder sacar los promedios de cada mes a partir de los datos de los archivos .csv.
Con el script ordenador.sh ordenamos los archivos en las carpetas correspondientes a su año.
Para finalmente usar hacer uso del python generateGraphic.py para graficar los graficos.
